# OESDK 0.0.16.3

## Status and error contract

- Added the signed 32-bit `OesdkStatus` public type.
- Added stable common status values from success through overflow and underflow.
- Added `OESDK_SUCCEEDED` and `OESDK_FAILED` result-test macros.
- Added `OesdkStatusName` and `OesdkStatusDescription` for diagnostics.
- Included the status API from `oesdk/kernel.h`.
- Added status-contract documentation and examples.
- Updated package revision and public revision macros to 0.0.16.3.
