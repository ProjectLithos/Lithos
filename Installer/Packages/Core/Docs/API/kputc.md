# `kputc`

**Category:** Console  
**Kind:** function  
**Header:** `<oesdk/console.h>`

Writes one character to the kernel console.

## Declaration

```c
void kputc(char character);
```

## Parameters

### `character`

Character to write.

## Returns

Nothing.

## Example

```c
kputc('A');
```
